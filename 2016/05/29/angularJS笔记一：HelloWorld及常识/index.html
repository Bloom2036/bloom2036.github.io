js 加载完之后,由 JIT 编译执行；

JIT编译器，英文写作Just-In-Time Compiler，中文意思是即时编译器。 JIT编译器能够将MSIL编译成为各种不同的机器代码，以适应对应的系统平台，最终使得程序在目标系统中得到顺利地运行。

Controller使用注意事项：

不要复用 Controller，一个控制器只负责一个视图(controller 中有重复代码请抽取出为一个 service) 不要在 Controller 中操作DOM
不要在 Controller 里做数据格式化 不要在 Controller 里做数据过滤操作

  • Controller 不要相互调用

$scope

$scope 是个 POJO
$scope 提供了一些工具方法 $watch()/$apply()
$scope 是表达式的执行环境(作用域)
$scope 是树形结构，与 DOM 标签平行
子 $scope 会继承父 $scope 的属性和方法
每一个 Aangular 应用只有一个根 $scope 对象(一般位于 ng-app)
$scope 可以传播事件，类似 DOM 事件，可向上或向下
可以通过 angular.element($0).scope() 进行调试(获取当前元素的 $scope)

$scope 生命周期
Creation -> Watcher registration -> Model mutation -> Mutation observation ->Scope destruction

angularjs 避免误区
表达式不要过于复杂，表达式数量不要过多 监听函数内不要有 DOM 操作，那样会显著降低性能
不能相互监听对方的修改的属性，避免交叉引用 ng 默认 TTL 为10次
*深拷贝式脏值检测会消耗更多内存(大型复杂Json数据)
