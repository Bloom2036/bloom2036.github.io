远程仓库

添加远程仓库

Step 1:
用户主目录下创建.ssh (#codeblock id_rsa 为私密钥匙,id_rsa.pub为公用钥匙#)

Step 2:
登陆 GitHub, set 选项,添加 SSH Key 粘贴 id_rsa.pub 内容。

Step 3:
GitHub新建仓库，其余默认设置，按照提示命令复制本地仓库。

克隆远程仓库

Step 1:
GitHub 创建一个新仓库；

Step 2:
输入 git clone git@github.com:<用户名>/<仓库名>.git 命令开始克隆仓库；
